# 📌 AI-Powered Paraphrasing Tool

## 🚀 Project Overview

This project is an AI-powered NLP system that performs: - Text
paraphrasing using T5 Transformer - Grammar checking using
LanguageTool - Evaluation using BLEU, ROUGE-L, and Semantic Similarity

------------------------------------------------------------------------

## 🧠 Features

-   Paraphrasing using Hugging Face T5 model
-   Grammar correction using LanguageTool
-   BLEU score evaluation
-   ROUGE-L score evaluation
-   Semantic similarity using Sentence Transformers
-   Interactive console mode

------------------------------------------------------------------------

## ⚙️ Installation

``` bash
pip install transformers sentencepiece torch
pip install nltk textblob rouge-score sentence-transformers language-tool-python
```

------------------------------------------------------------------------

## ▶️ How to Run

``` bash
python paraphraser.py
```

------------------------------------------------------------------------

## 📊 Evaluation Metrics

-   BLEU Score: Word overlap
-   ROUGE-L: Sequence similarity
-   Semantic Similarity: Meaning preservation
